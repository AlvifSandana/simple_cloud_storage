# Simple Cloud Storage
powered by ExpressJS

## HOW TO
### INSTALL
1. Clone this repo.
2. Install all dependencies with the following command:
```bash
npm install
```
3. Rename ```.env.example``` to ```.env``` and set your environment. 
4. Create new folder named ```data```
4. Run app:
```bash
# without nodemon
npm start
# with nodemon
npm run dev 
```
